# 画像一括リサイズツール (Image Batch Resizer)

PythonとTkinterを使用して作成した、シンプルで使いやすい画像リサイズ用のGUIアプリケーションです。
複数の画像を一括で選択し、指定した比率（%）でサイズを縮小・拡大することができます。

## 特徴 (Features)

* **直感的なGUI操作**: コマンドライン不要。ウィンドウ上から数値を入力して直感的に操作できます。
* **一括処理対応**: 複数の画像ファイルを一度に選択して、まとめてリサイズ可能です。
* **連続作業が可能**: 処理完了後もウィンドウが自動で閉じないため、設定を変えて何度でも続けて作業できます。
* **安全な保存処理**: 元の画像を上書きせず、指定した保存先フォルダに `resized_` というプレフィックスを付けて別名保存します。

## 必須環境 (Requirements)

* Python 3.x
* Pillow (Python Imaging Library)

## 使い方 (Usage)

1.  ターミナルまたはコマンドプロンプトで、スクリプトを実行します。
    ```bash
    python image_resizer.py
    ```
2.  起動したウィンドウで、リサイズしたい比率（例: `50` = 50%）を入力します。
3.  「画像を選択して実行」ボタンをクリックします。
4.  リサイズしたい画像ファイルを選択します（`Ctrl`キーやドラッグで複数選択可能）。
5.  リサイズ後の画像を保存するフォルダを選択します。
6.  処理が完了すると、完了メッセージが表示されます。
